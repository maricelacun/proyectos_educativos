Proyecto: Archivo BAT (Redes)

Los estudiantes desarrollarán un script de automatización en formato BAT para tareas de sistema operativo y redes.

Requisitos del Proyecto:

  • Script debe realizar al menos 4 tareas diferentes
  • Debe incluir manejo de parámetros
  • Debe incluir comentarios explicativos

Ejemplo de Código:

REM Script de automatización de red
REM Por: Grupo de Redes

@echo off
REM Verificar conexión a internet
ping -n 1 google.com > nul
if %errorlevel% equ 0 (
    echo Conexión a Internet: ACTIVA
) else (
    echo Conexión a Internet: INACTIVA
)

REM Listar recursos compartidos
net view

REM Crear respaldo de configuración
mkdir C:\Respaldo_Red
ipconfig /all > C:\Respaldo_Red\config_red.txt

Entregables:

  • Archivo BAT funcional
  • Documentación de uso
  • Diagrama de flujo del script
  • Capturas de pantalla de ejecución
